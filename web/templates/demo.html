How It Works

  1. User uploads an image of a fruit or vegetable
  2. Image is preprocessed to 224x224 pixels
  3. Deep learning model analyzes the image
  4. Model outputs classification: "Healthy" or "Rotten"
  5. Result is displayed to the user
